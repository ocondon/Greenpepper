Member Resolution

Given the following class:

class ActionExampleResolution
{
   public String personName() {return "personName";}
   public Person person() {return new Person();}

   public class Person
   {
      public String name() {return "person.name";}
      public String nick() {return "person.nick";}
   }
}

At first, a member of the fixture or SUT is searched, and then an expression build from the action specified:

rule for fixture member name resolution example
call return string()
person name personName
person nick person.nick

Rules for expression resolution order when not found as a member of the fixture or SUT are:

One token, taken as is (syntax is 'Class' # 'method') :

list of Object Graph Navigation Resolution Fixture   invoice
to string
#invoice

First tried with dot access at all spaces, and then with subset of dots placed at spaces:

list of Object Graph Navigation Resolution Fixture   invoice amount
to string
#invoiceamount
invoice#amount

With three tokens:

list of Object Graph Navigation Resolution Fixture   invoice line amount
to string
#invoicelineamount
invoiceline#amount
invoice#lineamount
invoice.line#amount
